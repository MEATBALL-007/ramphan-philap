# รำพันพิลาป — อินโฟกราฟิกวรรณคดีไทย

เว็บอินโฟกราฟิกหน้าเดียว นำเสนอวรรณคดีเรื่อง **รำพันพิลาป** ของพระสุนทรโวหาร (สุนทรภู่)
จัดทำเพื่อบอร์ดนิทรรศการภาษาไทย

🔗 **เว็บไซต์:** https://meatball-007.github.io/ramphan-philap/

## เนื้อหา
- ข้อมูลจำเพาะ (ผู้แต่ง / ปี พ.ศ. ๒๓๘๕ / วัดเทพธิดาราม / กลอนสุภาพ)
- เหตุจูงใจในการแต่ง (นิมิตฝัน → ประพันธ์ → ลาสิกขา)
- เนื้อเรื่องย่อและประเด็นสำคัญ
- บทกลอนเด่นตามต้นฉบับ
- คุณค่าของวรรณคดี ๓ ด้าน
- แบบทดสอบความเข้าใจ (คลิกตอบ พร้อมเฉลยและคะแนน)

เป็นไฟล์ `index.html` ไฟล์เดียว ไม่มี dependency เปิดได้ด้วยการดับเบิลคลิก
